# Full Stack Open 2025 - University of Helsinki

This repository contains my solutions to the **Full Stack Open** course. This course focuses on modern JavaScript-based web development, with a primary emphasis on building single-page applications with **React** and **Node.js**.

## 📂 Progress 
- [x] **Part 0:** Fundamentals of Web apps
- [x] **Part 1:** Introduction to React (Course info, Unicafe, Anecdotes)
- [ ] **Part 2:** Communicating with server

## 🚀 Key Learning Outcomes (Part 1)
* Component-based architecture and reusability.
* State management using the `useState` hook.
* Event handling and complex state logic.
* Debugging React applications.
